# Indiakart-ECommerce-Analytics
Built an end-to-end e-commerce analytics solution using Python, SQL, PostgreSQL, Power BI, and Excel with data cleaning, KPI analysis, SQL querying, and interactive dashboards.

# Project Overview

This project demonstrates an end-to-end data analytics workflow for an e-commerce business. It includes data cleaning, validation, exploratory data analysis, SQL-based querying, KPI development, and interactive dashboard creation.

The project simulates a real-world analytics solution for an online retail company and focuses on improving business decisions through data-driven insights.

# Business Objectives

- Analyze sales performance.
- Monitor customer behavior.
- Evaluate product performance.
- Track inventory levels.
- Analyze payment methods.
- Study return patterns.
- Build executive dashboards.
The objective of this project is to analyze sales performance, customer behavior, product trends, inventory management, and operational efficiency to support data-driven business decisions.

# Key Insights

- Electronics contributed the highest revenue.
- Premium customers generated the highest customer lifetime value.
- UPI was the most frequently used payment method.
- Damaged products accounted for the largest share of returns.
- Maharashtra generated the highest revenue.
- Some products consistently remained below reorder level.


# Business Recommendations

- Increase inventory for fast-moving products.
- Improve packaging quality to reduce returns.
- Expand loyalty programs for high-value customers.
- Monitor supplier performance monthly.
- Encourage digital payment adoption.
